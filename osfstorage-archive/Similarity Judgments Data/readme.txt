The file "rocks_similarity_120_X.txt" provides similarity judgments from participant X for the 120 rocks images. 
Participants 4, 19, 31, 37, 42, and 46 were excluded from the reported analyses for having low correlations with the other participants' judgments.
Each file contains 465 rows. Each row corresponds to one trial of the experiment. 
Column 1 = subtype of rock on left of screen
Column 2 = subtype of rock on right of screen
Column 3 = token number of rock on left of screen
Column 4 = token number of rock on right of screen
Column 5 = similarity judgment of rock pair
